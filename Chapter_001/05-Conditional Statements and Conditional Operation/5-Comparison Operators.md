- `>` greater than
- `>=` greater than or equal to
- `<` less than
- `<=` less than or equal to
- `==` equal to
- `!=` not equal to

```Java
int number = 55;

if (number != 0){
	System.out.println("The number is not equal to 0");
}

if (number >= 1000){
	System.out.println("The number is at least 1000");
}
// The number is not equal to 0
```
